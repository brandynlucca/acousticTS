# Spherical Bessel function of the first kind and its respective derivatives

Spherical Bessel function of the first kind and its respective
derivatives

## Usage

``` r
js(l, n)

jsd(l, n)

jsdd(l, n)
```

## Arguments

- l:

  A real integer or fractional order

- n:

  A real argument

## Value

Calculates the spherical Bessel function of the first kind (js) and both
its first (jsd) and second (jsdd) derivatives.

## References

Amos D.E., "AMOS, A Portable Package for Bessel Functions of a Complex
Argument and Nonnegative Order", http://netlib.org/amos

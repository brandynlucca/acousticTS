# Calculates the linear backscattering coefficient (sigma_bs) from the linear scattering length, f_bs.

Calculates the linear backscattering coefficient (sigma_bs) from the
linear scattering length, f_bs.

## Usage

``` r
sigma_bs(f_bs)
```

## Arguments

- f_bs:

  Linear scattering length (m), or related expression

## Value

Returns the linear backscattering coefficient that can be then converted
into TS.

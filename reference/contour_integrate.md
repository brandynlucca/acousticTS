# Numerical integration via adaptive quadrature of complex values

Numerical integration via adaptive quadrature of complex values

## Usage

``` r
contour_integrate(integral, x, y)
```

## Arguments

- integral:

  Input integration function that is indexed so that it can be used
  within [`base::apply`](https://rdrr.io/r/base/apply.html).

- x:

  Indexing argument for multi-row objects

- y:

  Indexing argument for multi-column objects

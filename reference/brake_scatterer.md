# Support function for bending scatterer body shape scatterer object

Support function for bending scatterer body shape scatterer object

## Usage

``` r
brake_scatterer(object, radius_curvature, mode = "ratio")
```

## Arguments

- object:

  Scatterer-class object

- radius_curvature:

  Radius of curvature that can be parameterized either as a ratio
  relative to body length or actual measurement

- mode:

  Either "ratio" or "measurement"

# Creates a sphere.

Creates a sphere.

## Usage

``` r
sphere(radius_body, n_segments, diameter_units )
```

## Arguments

- radius_body:

  Object radius (m).

- n_segments:

  Number of segments to discretize object shape. Defaults to 1e2
  segments.

- diameter_units:

  Default is "m" for meters

## Value

Creates position vector for a spherical object of a defined radius.

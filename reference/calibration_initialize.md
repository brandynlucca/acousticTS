# Initialize CAL-class object for modeling.

Initialize CAL-class object for modeling.

## Usage

``` r
calibration_initialize(
  object,
  frequency,
  sound_speed_sw = 1500,
  density_sw = 1026
)
```

## Arguments

- object:

  CAL-class object.

- frequency:

  Frequency (Hz).

- sound_speed_sw:

  Seawater sound speed.

- density_sw:

  Seawater density.

# Calculates the theoretical TS of a fluid-like scatterer at a given frequency using the stochastic distorted Born wave approximation (DWBA) model.

Calculates the theoretical TS of a fluid-like scatterer at a given
frequency using the stochastic distorted Born wave approximation (DWBA)
model.

## Usage

``` r
SDWBA(object)
```

## Arguments

- object:

  FLS-class scatterer.

## References

Demer, D.A., and Conti, S.G. 2003. Reconciling theoretical versus
empirical target strengths of krill: effects of phase variability on the
distorted-wave Born approximation. ICES J. Mar. Sci., 60: 429-434.

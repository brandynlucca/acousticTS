# Support rotation function for KRM (swimbladder)

Support rotation function for KRM (swimbladder)

## Usage

``` r
bladder_rotation(sum_rpos, rpos, theta, k_length)
```

## Arguments

- sum_rpos:

  Summed position matrix

- rpos:

  Position matrix

- theta:

  Orientation angle

- k_length:

  Length of wavenumber vector

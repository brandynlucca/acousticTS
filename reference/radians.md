# Convert angular measurements from degrees to radians.

Convert angular measurements from degrees to radians.

## Usage

``` r
radians(x)
```

## Arguments

- x:

  A real value in degrees

## Value

Angle in radians.

## Examples

``` r
orientation <- 90 # degrees
radians(orientation) # this should return a value equal to pi / 2 radians
#> [1] 1.570796
```

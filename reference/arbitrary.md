# Creates arbitrary body shape from user inputs

Creates arbitrary body shape from user inputs

## Usage

``` r
arbitrary(x_body, y_body, z_body, radius_body, length_units = "m")
```

## Arguments

- x_body:

  x-axis (m)

- y_body:

  y-axis (m)

- z_body:

  z-axis (m)

- radius_body:

  Radius (m)

- length_units:

  Units for body length. Defaults to meters: "m"

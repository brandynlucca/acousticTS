# Reforge FLS-class object.

Reforge FLS-class object.

## Usage

``` r
# S4 method for class 'FLS'
reforge(
  object,
  length,
  radius,
  length_radius_ratio_constant = TRUE,
  n_segments
)
```

## Arguments

- object:

  FLS-class object.

- length:

  New body length resize.

- radius:

  New radius size

- length_radius_ratio_constant:

  Keep length-to-radius ratio based on new length

- n_segments:

  New number of segments

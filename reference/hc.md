# Cylindrical Bessel (Hankel) function of the third kind and its derivatives

Cylindrical Bessel (Hankel) function of the third kind and its
derivatives

## Usage

``` r
hc(l, n)

hcd(l, n)

hcdd(l, n)
```

## Arguments

- l:

  A real integer or fractional order

- n:

  A real argument

## Value

Calculates the cylindrical Bessel function of the first kind (\\H_v\\)
and its respective derivatives.

## References

Amos D.E., "AMOS, A Portable Package for Bessel Functions of a Complex
Argument and Nonnegative Order", http://netlib.org/amos

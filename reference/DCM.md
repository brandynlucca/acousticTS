# Calculates the theoretical TS of a target using the deformed cylinder model (DCM).

Calculates the theoretical TS of a target using the deformed cylinder
model (DCM).

## Usage

``` r
DCM( object )
```

## Arguments

- object:

  FLS-class scatterer.

## Value

Target strength (TS, dB re. 1 \\m^2\\) of a FLS-class object. \#'
@references Stanton, T.K., Chu, D., and Wiebe, P.H. 1998. Sound
scattering by several zooplankton groups. II. Scattering models. Journal
of the Acoustical Society of America, 103(1), 236-253.

# Calculates the theoretical TS of a fluid sphere using an exact modal series solution proposed by Andersen (1950).

Calculates the theoretical TS of a fluid sphere using an exact modal
series solution proposed by Andersen (1950).

## Usage

``` r
MSS_anderson(object)
```

## Arguments

- object:

  GAS- or SBF-class object.

## Value

Target strength (TS, dB re: 1 m^2)

## Details

Calculates the theoretical TS of a fluid sphere at a given frequency
using an exact modal series solution.

## References

Anderson, V.C. (1950). Sound scattering from a fluid sphere. Journal of
the Acoustical Society of America, 22, 426-431.

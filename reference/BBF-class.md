# Backboned fish (BBF) object/class.

A S4 class that stores a fluid-like flesh component together with an
explicit elastic backbone component for swimbladder-less fish
applications where those two structures should remain separate. The body
and backbone each retain their own geometry, orientations, and material
properties. This class is intended to support composite
flesh-plus-backbone model development and future extensions where
additional internal structures may need to be carried alongside the
body.

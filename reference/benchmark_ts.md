# Benchmark model outputs from Jech et al. (2015)

A packaged list of benchmark target-strength spectra for the canonical
sphere, shell-sphere, finite-cylinder, and prolate-spheroid comparison
cases used throughout the package validation workflow. The stored values
follow the benchmark definitions assembled by Jech et al. (2015) and
distributed in the `echoSMs` reference resources.

## Usage

``` r
data(benchmark_ts)
```

## Format

An object of class `list` of length 2.

## Source

`echoSMs` benchmark resources, especially the target-definition and Jech
benchmark tables archived at
<https://github.com/ices-tools-dev/echoSMs>.

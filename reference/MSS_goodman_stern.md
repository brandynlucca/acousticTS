# Calculates the theoretical TS of an elastic-shelled sphere using the modal series solution from Goodman and Stern (1962).

Calculates the theoretical TS of an elastic-shelled sphere using the
modal series solution from Goodman and Stern (1962).

## Usage

``` r
MSS_goodman_stern(object)
```

## Arguments

- object:

  GAS- or SBF-class object.

## Value

Target strength (TS, dB re: 1 m^2)

## Details

Calculates the theoretical TS of an elastic-shelled sphere using an
exact modal series solution

## References

Goodman, R.R., and Stern, R. (1962). Reflection and transmission of
sound by elastic spherical shells. The Journal of the Acoustical Society
of America, 34, 338-344.

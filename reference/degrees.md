# Convert angular measurements from radians to degrees

Convert angular measurements from radians to degrees

## Usage

``` r
degrees(x)
```

## Arguments

- x:

  A real value in radians

## Value

Angle in degrees

## Examples

``` r
orientation <- pi / 2 # radians
degrees(orientation) # this should return a value equal to 90 degrees
#> [1] 90
```
